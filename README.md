# **AIO Chatbot – AI-Powered Conversational Assistant**

## **Project Purpose & Key Features**  

AIO Chatbot is an **AI-powered multi-functional assistant** designed to handle diverse user interactions. It integrates **Natural Language Processing (NLP)**, **mathematical computation**, **reminder management**, and **real-time weather updates**, making it a versatile tool for both casual and productivity-driven conversations.  

### **Key Objectives:**  
-  **Intelligent Conversations** – Uses **Groq API** to generate human-like responses.  
-  **Smart Task Management** – Allows users to **set and retrieve reminders**.  
-  **Mathematical Computation** – Processes calculations directly through chat.  
-  **Real-time Weather Updates** – Fetches current weather using **OpenWeatherMap API**.  
-  **Calendar Assistance** – Provides date and weekday insights.  
-  **Seamless User Interaction** – Built with **React.js (Frontend)** and **Flask (Backend)**.  

---

## **Core Technologies Used**  

### ** AI & Backend**  
 **Groq API** – NLP-based AI for intelligent responses.  
 **Flask (Python)** – Manages chatbot logic and API integrations.  
 **MySQL Database** – Stores reminders and user interactions.  
 **OpenWeatherMap API** – Fetches real-time weather updates.  

### **🎨 Frontend & UI**  
 **React.js** – Creates an interactive user experience.  
 **Bootstrap** – Enhances styling and responsiveness.
